# 疑問詞

| 疑問詞                           | 意味               | 例文1                                                          | 例文2                                                |
| -------------------------------- | ------------------ | -------------------------------------------------------------- | ---------------------------------------------------- |
| qué                              | 何                 | ¿Qué haces?（何してるの？）                                    | ¿Qué libro lees?（どんな本を読んでいるの？）         |
| quién, quiénes                   | 誰, 誰たち         | ¿Quién es ella?（彼女は誰？）                                  | ¿Quiénes son ellos?（彼らは誰？）                    |
| dónde                            | どこ               | ¿Dónde vives?（どこに住んでるの？）                            |
| cuándo                           | いつ               | ¿Cuándo es tu cumpleaños?（誕生日はいつ？）                    |
| cómo                             | どのように, どう   | ¿Cómo estás?（元気？）                                         | ¿Cómo llegas al trabajo?（どうやって仕事に行くの？） |
| cuál, cuáles                     | どれ, どれら       | ¿Cuál es tu color favorito?（好きな色はどれ？）                | ¿Cuáles son tus libros?（あなたの本はどれ？）        |
| cuánto, cuánta, cuántos, cuántas | いくつ, どれくらい | ¿Cuánto cuesta?（いくらですか？）                              | ¿Cuántas hermanas tienes?（姉妹は何人？）            |
| por qué                          | なぜ               | ¿Por qué estudias español?（なぜスペイン語を勉強しているの？） |
| para qué                         | 何のために         | ¿Para qué sirve esto?（これは何のために使うの？）              |
| de dónde                         | どこから           | ¿De dónde eres?（どこの出身？）                                |
| adónde                           | どこへ             | ¿Adónde vas?（どこへ行くの？）                                 |
